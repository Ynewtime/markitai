58@dataclass 

59class FetchResult: 

60 """Result of a URL fetch operation. 

61 

62 Supports multi-source content for URL fetching: 

63 - content: Primary markdown content (best available) 

64 - static_content: Content from static/jina fetch (pure text) 

65 - browser_content: Content from browser fetch (rendered page) 

66 - screenshot_path: Full-page screenshot (visual reference) 

67 

68 For LLM processing, all three sources can be provided: 

69 1. static_content - Clean text, reliable but may miss JS content 

70 2. browser_content - Rendered content, includes JS but may have noise 

71 3. screenshot - Visual reference for layout/structure 

72 """ 

73 

74 content: str # Primary markdown content (best available) 

75 strategy_used: str # Actual strategy used (static/browser/jina) 

76 title: str | None = None # Page title if available 

77 url: str = "" # Original URL 

78 final_url: str | None = None # Final URL after redirects 

79 metadata: dict = field(default_factory=dict) # Additional metadata 

80 cache_hit: bool = False # Whether result was served from cache 

81 screenshot_path: Path | None = None # Path to captured screenshot (if any) 

82 # Multi-source content for enhanced LLM processing 

83 static_content: str | None = None # Content from static fetch 

84 browser_content: str | None = None # Content from browser fetch
