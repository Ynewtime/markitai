1"""Resolver layer: structured extractor orchestration above root selection. 

2 

3The resolver layer sits between site-specific extractor lookup and the generic 

4pipeline extraction. It provides a typed contract (``ResolvedPage``) that 

5an extractor's optional ``resolve()`` method may return. 

6 

7Design rules: 

8- ``ResolvedPage`` lives here, NOT in ``FetchResult`` or fetch-layer types. 

9- A resolver may return ``content_root`` (a Tag) or ``content_html`` (a string). 

10 It must NEVER return final Markdown. 

11- If no extractor implements ``resolve()``, or no extractor matches the URL, 

12 ``resolve_page()`` returns ``None`` and the generic pipeline remains the 

13 fallback. 

14- Metadata overrides from ``ResolvedPage.metadata_overrides`` are merged over 

15 the generic metadata extracted by ``extract_metadata()``. 

16""" 

17

31@dataclass(slots=True) 

32class ResolvedPage: 

33 """Output of a site-specific resolver. 

34 

35 A resolver may populate either ``content_root`` (a parsed Tag) or 

36 ``content_html`` (a raw HTML string). It must NOT set final Markdown 

37 in either field — the pipeline is responsible for the HTML→Markdown 

38 conversion step. 

39 

40 Attributes: 

41 content_root: Parsed BeautifulSoup Tag representing the content root. 

42 Mutually exclusive with ``content_html`` (caller may set either). 

43 content_html: Raw HTML string of the primary content. 

44 Mutually exclusive with ``content_root``. 

45 metadata_overrides: Mapping of metadata fields to override (e.g. 

46 ``{"title": "Custom Title"}``). Merged over generic metadata. 

47 semantic: Optional semantic models derived during resolution (e.g. 

48 a conversation thread). 

49 diagnostics: Internal debug information populated during resolution. 

50 """ 

51 

52 content_root: Tag | None = None 

53 content_html: str | None = None 

54 metadata_overrides: dict[str, Any] = field(default_factory=dict) 

55 semantic: SemanticExtraction | None = None 

56 diagnostics: dict[str, Any] = field(default_factory=dict)

59def resolve_page( 

60 html: str, 

61 url: str, 

62 *, 

63 resolver: object | None = None, 

64) -> ResolvedPage | None: 

65 """Attempt structured resolution of a page via a site-specific extractor. 

66 

67 Parses ``html``, finds a matching extractor (or uses the provided 

68 ``resolver``), and calls its ``resolve()`` method if present. The result 

69 is validated to ensure it is a ``ResolvedPage`` and not raw Markdown. 

70 

71 Args: 

72 html: Raw HTML source of the page. 

73 url: Canonical URL of the page (used for extractor lookup). 

74 resolver: Optional extractor override. When supplied, skips the 

75 extractor registry lookup and uses this object directly. 

76 

77 Returns: 

78 A ``ResolvedPage`` if a resolver matched and returned structured 

79 content, or ``None`` when falling back to the generic pipeline. 

80 

81 Raises: 

82 TypeError: If the resolver's ``resolve()`` method returns something 

83 other than a ``ResolvedPage`` (e.g. a Markdown string). 

84 """ 

85 extractor = resolver 

86 if extractor is None: 

87 from markitai.webextract.extractors.registry import find_extractor 

88 

89 extractor = find_extractor(url) 

90 

91 if extractor is None: 

92 return None 

93 

94 resolve_fn = getattr(extractor, "resolve", None) 

95 if resolve_fn is None or not callable(resolve_fn): 

96 return None 

97 

98 soup = parse_html(html) 

99 raw_result = resolve_fn(soup, url) 

100 

101 # Validate: resolver must not return final Markdown 

102 _validate_resolver_result(raw_result, extractor) 

103 

104 return raw_result # type: ignore[return-value] 

105

107def _validate_resolver_result(result: object, extractor: object) -> None: 

108 """Validate that a resolver did not return Markdown or another illegal type. 

109 

110 Args: 

111 result: The raw return value from the extractor's ``resolve()`` method. 

112 extractor: The extractor object (used in error messages). 

113 

114 Raises: 

115 TypeError: If ``result`` is not a ``ResolvedPage`` instance. 

116 """ 

117 if isinstance(result, ResolvedPage): 

118 return 

119 

120 extractor_name = getattr(extractor, "name", repr(extractor)) 

121 raise TypeError( 

122 f"Extractor '{extractor_name}'.resolve() must return a ResolvedPage, " 

123 f"not {type(result).__name__!r}. " 

124 "Returning final Markdown from a resolver is not allowed; " 

125 "the pipeline handles HTML→Markdown conversion." 

126 )
