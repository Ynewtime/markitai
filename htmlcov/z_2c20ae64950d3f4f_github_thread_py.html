18class GitHubThreadExtractor: 

19 """Extract GitHub issue or pull request threads into a ConversationThread. 

20 

21 Targets github.com issue and pull request URLs. Parses the issue body as 

22 the ``main_item`` and collects subsequent comments as ``items`` in the 

23 thread, discarding sidebar content (Assignees, Labels, Milestone, etc.). 

24 

25 GitHub HTML structure targeted: 

26 - Issue title: ``<bdi class="js-issue-title">`` or 

27 ``<h1 class="gh-header-title"><span>`` 

28 - Issue/comment bodies: ``<div class="comment-body">`` 

29 - Author per block: ``<a class="author">username</a>`` 

30 - Timestamp per block: ``<relative-time datetime="...">`` 

31 - Sidebar: ``<div class="Layout-sidebar">`` (excluded) 

32 """

110 

111 return ResolvedPage( 

112 content_html=content_html, 

113 metadata_overrides=metadata_overrides, 

114 semantic=semantic,

118 "extractor_name": self.name,
